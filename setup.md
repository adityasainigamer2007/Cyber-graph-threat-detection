# Setup Instructions for Cyber Graph Threat Detection

## Prerequisites
1. **Operating System**: Ensure you are running on a compatible OS (Windows, macOS, Linux).
2. **Node.js**: Install Node.js (version 14 or higher) from [nodejs.org](https://nodejs.org/).
3. **Package Manager**: Make sure you have npm (Node Package Manager) installed with Node.js.
4. **Git**: Install Git from [git-scm.com](https://git-scm.com/) to clone the repository.

## Installation Steps
1. **Clone the Repository**: Open terminal or command prompt and run:
   ```bash
   git clone https://github.com/adityasainigamer2007/Cyber-graph-threat-detection.git
   ```
2. **Navigate to Project Directory**: 
   ```bash
   cd Cyber-graph-threat-detection
   ```
3. **Install Dependencies**: Run the following command to install the required libraries:
   ```bash
   npm install
   ```
4. **Build the Project** (if applicable): 
   ```bash
   npm run build
   ```

## Running the Application
1. **Start the Server**: Execute the following command to run the application:
   ```bash
   npm start
   ```
   The server should start, and you will see messages indicating it is running.

## Accessing the Web Interface
- Open your web browser and go to `http://localhost:3000` (replace `3000` with your configured port if different).
- You should see the Cyber Graph Threat Detection web interface. 

If you face any issues, please refer to the troubleshooting section in the repository issues page.
